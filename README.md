# HEKAX

## AI Infrastructure for Modern Business Communication

HEKAX develops multi-layered AI systems designed to improve how service-based businesses communicate, respond, and operate. The platform integrates voice intelligence, messaging, workflow automation, and industry-specific reasoning into a unified architecture.

---

## Core Pillars

**Voice AI**  
Real-time call intelligence and automated reception.

**SLM Intelligence**  
Small, specialized language models for fast inference.

**Automation OS**  
Multi-tenant communication and decision systems built for scale.

## Architectural Principles

### Multi-World Operating Model

HEKAX is structured as a multi-world system:

- **Calls World** — Voice operations and receptionist intelligence
- **Messages World** — Unified messaging and Smart Inbox
- **Live Control World** — Real-time monitoring, SLA, and queue management

These worlds operate cohesively under the HEKAX orchestration layer.

### Codex Decision Engine

A modular AI policy engine that performs:

- Multi-vertical reasoning
- Labeling and classification
- Priority and urgency detection
- Action routing and automation

Codex is designed to support diverse industries including legal, medical, HVAC, BPO, and SaaS.

### SLM Intelligence Layer

Custom-tuned small language models optimized for:

- Low latency
- Real-world workflows
- Domain-specific reasoning
- Hybrid routing between fast and accurate models

### Voice AI Stack

Real-time processing pipeline for:

- Speech-to-text
- Text-to-speech
- Voice activity detection
- Segment analysis
- Caller intent extraction

### Event Mesh Architecture

A distributed system using:

- Redis Streams
- Consumer groups
- Durable logs
- WebSocket broadcasting
- State machines
- PostgreSQL storage
